# YT Downloader

Free video downloader website. Download from YouTube, Facebook, Instagram, Twitter, TikTok in HD quality.

## Features
- Multiple quality options (360p to 4K)
- Audio extraction (MP3)
- Works with 1000+ sites
- Clean, responsive UI
- No registration required

## Setup

### 1. Install Node.js
Download from https://nodejs.org (LTS version)

### 2. Install yt-dlp
```bash
pip install yt-dlp
```

### 3. Run
```bash
npm install
npm start
```

Open http://localhost:3000

## Tech Stack
- Node.js + Express
- yt-dlp (video processing)
- Vanilla JavaScript frontend
